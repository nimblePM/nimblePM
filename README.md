# NimblePM - OpenProject Fork

NimblePM is a customized fork of OpenProject v16.3.2, tailored for enhanced project management capabilities.

## About

This repository contains our production deployment of OpenProject with custom configurations and enhancements.

## Base Version

- OpenProject 16.3.2
- Ruby-based project management software
- PostgreSQL database backend

## Deployment

The application is deployed on nimblepm.ca with:
- Nginx as reverse proxy
- PostgreSQL database
- Cloudflare for SSL/CDN
